**SO_LONG**

**So_long** is a project at **42 school**.

The aim is to create a basic 2D game where the player must collect all items  on the map before reaching the exit. This project is built using MiniLibX, a tiny graphic library.

**Controls**
* **Movement**: Use **WASD** to move the character
* **Quit**: Close the window by clicking on the **X** button or pressing the **ESC** key
* **Objective**: The game ends once you have collected all the objects on the map and reached the exit
